`starlight-package-managers` supports the following package managers:

- [npm](https://www.npmjs.com)
- [yarn](https://yarnpkg.com)
- [pnpm](https://pnpm.io)
- [bun](https://bun.sh)
- [ni](https://github.com/antfu/ni)

## Defaults

By default, `starlight-package-managers` will use `npm`, `yarn` and `pnpm`.

<Preview>

```mdx title="src/content/docs/example.mdx"
import { PackageManagers } from 'starlight-package-managers'

<PackageManagers pkg="astro" />
```

<PackageManagers slot="preview" pkg="astro" />

</Preview>

## Customization

You can customize the package managers using the `pkgManagers` prop.

<Preview>

```mdx title="src/content/docs/example.mdx" 'pkgManagers={['npm', 'bun']}'
import { PackageManagers } from 'starlight-package-managers'

<PackageManagers pkg="astro" pkgManagers={['npm', 'bun']} />
```

<PackageManagers slot="preview" pkg="astro" pkgManagers={['npm', 'bun']} />

</Preview>

:::note
If only a single package manager is specified, the command will be displayed with no tabs.

<Preview>

```mdx title="src/content/docs/example.mdx"
import { PackageManagers } from 'starlight-package-managers'

<PackageManagers pkg="astro" pkgManagers={['pnpm']} />
```

<PackageManagers slot="preview" pkg="astro" pkgManagers={['pnpm']} />

</Preview>

:::

## Global Customization

Having to specify the `pkgManagers` prop every time can be tedious. To avoid this, you can create a custom Astro component wrapping `starlight-package-managers`:

```astro title="src/components/AllPackageManagers.astro"
---
import {
  PackageManagers,
  type PackageManagersProps,
} from 'starlight-package-managers'

type Props = PackageManagersProps
---

{/* Customize the package managers to use. */}
<PackageManagers
  pkgManagers={['npm', 'yarn', 'pnpm', 'bun', 'ni']}
  {...Astro.props}
/>
```

Now you can use the custom component instead of `starlight-package-managers`:

<Preview>

```mdx title="src/content/docs/example.mdx"
import AllPackageManagers from '@/components/AllPackageManagers.astro'

<AllPackageManagers pkg="astro" />
```

<AllPackageManagers slot="preview" pkg="astro" />

</Preview>
